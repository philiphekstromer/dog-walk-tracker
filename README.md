# Dog walk tracker

A simple app to remind you when it's time to walk the dog. <a href="https://phieksmig.github.io/dog-walk-tracker">Use the app here</a>


<!-- ABOUT THE PROJECT -->
## About The Project

This is a simple app I made to help me remember when I need to walk the dog. WHen you come home, you press the button, select how long the walk was and a countdown starts showing the time untill the next walk. 


### Built With

React, ChatGPT & Github Copilot
